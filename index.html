Affect-Sampler by benmoran

Experience sampling app for Android

This is a personal informatics project for Android, intended to help you perform "experience sampling": quickly capture time-series data about your mood at configurable random intervals through the day.

Dependencies

Using http://code.google.com/p/achartengine/ library for graphing (included)

Install

Client - Download the .apk file and install to an Android device (tested on HTC Hero). Ensure that "Allow install of non-Market applications" is checked in Settings->Application. Server - It's configured to use the existing server at https://affectsampler.appspot.com, but you can edit the supplied server code and configure your own server on Google AppEngine.
